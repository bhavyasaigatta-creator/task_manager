# 📝 Task Manager App

A full-stack Task Manager application built using **React, Node.js, Express, and MongoDB**.
Users can **sign up, log in, and manage their personal tasks** with a clean and responsive UI.

---

## 🚀 Live Demo

🔗 Frontend: https://task-manager-tau-sand-41.vercel.app/  
🔗 Backend: https://task-manager-backend-lmaf.onrender.com

---

## ✨ Features

* 🔐 User Authentication (Signup & Login)
* ➕ Add Tasks
* 🗑️ Delete Tasks
* ✅ Mark Tasks as Completed / Undo
* 👤 User-specific tasks (each user sees only their tasks)
* 💾 Data stored in MongoDB
* 🌐 Fully deployed (Vercel + Render)

---
